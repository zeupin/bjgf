# bjgf

一个产品的报价样式。

![](demo1.png)

## 层次结构

```
.bjgf                     主类
    .bjgf-h1              大标题
    .bjgf-h2              副标题
    .bjgf-desc            描述文字
    hr.bjgf-line          横线
    ul.bjgf-list          功能列表
        li                功能点1
        li                功能点2

工具：
.bjgf-price               价格（1.5倍大，红色）
.bjgf-btn                 按钮
em或.em                   重点文字（放大1.5倍）
```
